Command line interface

Some operations can be performed by the command line utility jl.
Optionally, you can set user and password as environment variables to avoid passing them as command line arguments:

export JLOGIC_USER=user
export JLOGIC_PASSWORD=password
To see all options type:
cd jlogic-<version>/bin
./jl -h

Installing a plugin

Plugins can be installed as shown below:

./jl -u user -p password install /path/to/jar/file
Example:
./jl -u foo -p bar install /data/Downloads/json-1.0.0.jar
User must have administration privileges to install or uninstall a plugin.
