Command line tools

Some operations can be performed in the terminal.
Optionally, you can set to environmental variables with user and password to avoid passing them each time as command line arguments:

export JLOGIC_USER=user
export JLOGIC_PASSWORD=password
Currently available tools are:
  • plugin For plugin management

Installing plugins

Plugins can be installed using the plugin script.

cd jlogic-<version>/bin
./plugin --user user --password password --install /path/to/jar/file
Example:
./plugin -u foo -p bar -i /data/Downloads/json-1.0.0.jar
User must have administration privileges to install or uninstall a plugin.
To see other options type
./plugin --help
