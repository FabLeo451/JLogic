Getting started

Requirements

  • Linux based operating system
  • JDK 1.7 or later
  • Maven 3.2 or later

Starting server

Download and unzip the package. Then start the server:
tar -vxf jlogic-<version>.tar.xz
cd jlogic-<version>
bin/jlogic
Go to https://localhost:8443/home with your browser.
To start on a different port:
bin/jlogic --server.port=myport
Default Maven home is /usr. If your Maven installation is somewhere else provide the path when launching:
bin/jlogic --maven.home=/opt/apache-maven-3.6.3

First access

Log in with credentials below:
  • User: admin
  • Password: admin
It's recommended that you change the password as soon as possible.
User admin has grants to create other users.
