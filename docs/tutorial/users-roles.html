Users and roles

A default account admin/admin is created on first start. Other users can be created by user admin.
It's recommended that you change the password as soon as possible.

Every user has a role that defines permitted and forbidden tasks. The roles are:

  • ADMIN can do anything, included system tasks
  • EDITOR can edit objects end execute programs and APIs
  • VIEWER can only view blueprints, properties and APIs
  • USER can only execute REST APIs
